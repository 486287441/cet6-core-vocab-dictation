## 本地运行

**方式一（推荐）**：

| 系统 | 操作 |
| --- | --- |
| Windows | 双击 `start.bat` |
| macOS / Linux | 终端执行 `chmod +x start.sh && ./start.sh` |

默认端口 **8888**，会自动打开浏览器。自定义端口：`PORT=3000 ./start.sh`

**方式二（命令行）**：

```bash
cd "项目根目录"
python3 -m http.server 8888
```

浏览器打开 `http://localhost:8888/`（勿用 `file://`）。
