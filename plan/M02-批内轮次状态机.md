# M02 — 批内轮次状态机

技术依据：见 `plan/技术方案.md` §4.2。

## 目标

纯逻辑模块：实现本批 50 词内的轮次复习、通关判定、曾「不记得」记录（与 UI 解耦，便于单测或控制台验证）。

## 实现要点

1. `js/state.js` 导出 `createBatchSession(batchWords)`。
2. API 建议：
   - `getCurrentWord()`
   - `judge(remember: boolean)` → 更新队列、`everForgotten`
   - `canAdvance()` / `advance()`（配合闪卡「下一张」）
   - `isBatchComplete()`
   - `getEverForgottenList()`
3. 规则严格对齐 PRD：记得→本轮移除；不记得→入复习池；下一轮只刷上轮不记得词。

## 验收步骤

| # | 操作 | 期望 |
| --- | --- | --- |
| 1 | 用脚本或测试页：50 词全判「记得」 | 一轮后 `isBatchComplete() === true` |
| 2 | 50 词全判「不记得」 | 需多轮；最终全改「记得」后通关 |
| 3 | 混合：前 25 记得、后 25 不记得，再第二轮全记得 | 第二轮结束后通关 |
| 4 | `getEverForgottenList()` | 包含所有曾判不记得的词（含后来改记得的） |
