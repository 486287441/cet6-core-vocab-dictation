# M03 — 闪卡页与键盘

技术依据：见 `plan/技术方案.md` §5（键盘行）、`需求.md` §4。

## 目标

闪卡视图：仅英文 → 判词展开中文 → 空格下一张；完整键盘与改判、轻提示。

## 实现要点

1. `js/views/flashcard.js` 绑定 `state.js`。
2. UI：大字号英文居中；中文区默认隐藏，`revealed` 后淡入+位移（CSS）。
3. `main.js` 全局 `keydown`：`a/A/ArrowLeft`、`d/D/ArrowRight`、`Space`（preventDefault 当需要）。
4. 未判按空格 → `ui/toast.js` 显示「先按 A 或 D」。
5. 已出中文：A/D 改判并更新 UI；空格 → `advance()`。
6. `ui/progress.js`：本批第几张 / 本轮队列剩余数。

## 验收步骤

| # | 操作 | 期望 |
| --- | --- | --- |
| 1 | 进入第 1 批闪卡 | 仅英文，无中文 |
| 2 | 按 D | 中文出现在英文下方，有过渡 |
| 3 | 未判按空格 | Toast 提示，不换卡 |
| 4 | 出中文后按空格 | 下一张，中文隐藏 |
| 5 | 出中文后按 A（改判） | 状态变为不记得，中文仍可见 |
| 6 | ←/→ 与 A/D 行为一致 | 同上 |
