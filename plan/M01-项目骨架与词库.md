# M01 — 项目骨架与词库

技术依据：见 `plan/技术方案.md` §1–§2、§4.1。

## 目标

可运行的静态站点骨架，能正确加载并分批展示 `words.csv`。

## 实现要点

1. 创建 `index.html`：三个空容器 `#view-flashcard` / `#view-batch-stats` / `#view-celebrate`（先隐藏）。
2. `css/app.css`：浅色 CSS 变量、`.page` 居中、系统字体栈。
3. `js/words.js`：
   - `loadWords(): Promise<Word[]>`
   - `getBatch(words, batchIndex): Word[]`
   - `getTotalBatches(words): number`（`ceil(n/50)`）
4. `js/main.js`：DOMContentLoaded → loadWords → 控制台或临时页显示「共 N 词、M 批」。

## 验收步骤

| # | 操作 | 期望 |
| --- | --- | --- |
| 1 | 项目根 `python -m http.server 8080`，打开 `/` | 无控制台报错 |
| 2 | 查看临时调试输出或页眉 | 词数 ≈ 300+，批次数 = `ceil(词数/50)` |
| 3 | 检查第 1 批第 1 词、最后一批最后一词 | 与 `words.csv` 首尾一致 |
| 4 | `file://` 直接打开 | **允许失败**（document 说明需 HTTP） |
