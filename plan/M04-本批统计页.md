# M04 — 本批统计页

技术依据：`需求.md` §3 统计页、`技术方案.md` §4.2 `everForgotten`。

## 目标

本批通关后展示：本轮曾「不记得」词表（英+中）、本批用时；按钮进入下一批。

## 实现要点

1. `js/views/batchStats.js`。
2. 本批用时 = `Date.now() - batchStartedAt`（进入本批时记录）。
3. 列表为空时文案如「本轮一次全记住，太棒了」。
4. 「下一批」：`currentBatchIndex++`，新建 `BatchSession`；若已是最后一批且通关 → 路由庆祝（M05）。

## 验收步骤

| # | 操作 | 期望 |
| --- | --- | --- |
| 1 | 故意标记若干词「不记得」后通关 | 列表含这些词及中文 |
| 2 | 全记得通关 | 空状态文案友好 |
| 3 | 显示用时 | 格式如 `分:秒`，大于 0 |
| 4 | 点下一批 | 进入下一批闪卡，进度批号 +1 |
